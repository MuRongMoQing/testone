# 使用双免密汇率源与本地缓存

汇率访问通过可替换的提供方接口实现，首版零配置使用 ExchangeRate.fun，失败时自动切换至 USDConverter，再失败时使用本地最后成功缓存；不把欧洲央行每日参考价作为自动后备。因免费源按小时更新，外币入账只接受报价时间不超过 90 分钟的数据，并永久保存提供方、报价时间、取得时间、原币金额、汇率和人民币金额；没有合格报价时外币操作保持待执行且不改变库存或账册，人民币操作不受影响。

模拟提供方只验证切换、缓存和失败语义，不能替代目标部署网络对 ExchangeRate.fun 与 USDConverter 的真实连接、响应字段和报价时间验证。任一真实检查失败必须如实报告，不能改称供应商已验证，也不能未经新决策改用 ECB 或其他来源。金额与汇率的具体数据库边界由 ADR 0038 统一约束。
