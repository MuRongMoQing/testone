# 首版只实现复式库存会计子账

系统为已生效的入库、取出及跨仓调拨自动生成借贷平衡、不可覆盖的会计分录，并据此计算库存估值、销售收入、销售成本和毛利润；人民币统一账册与外币账册共享同一业务关联。入库借记库存资产、贷记采购暂记；取出借记销售暂记、贷记销售收入，同时借记销售成本、贷记库存资产；同仓库内的库内移位不产生金额分录，汇率舍入差额进入独立科目。

跨仓分批发运时借记在途库存、贷记发出仓库存资产；分批收货时借记接收仓库存资产、贷记在途库存。调拨全程沿用原批次的原币成本、既有锁定汇率和人民币成本，不重新获取汇率，也不产生销售收入、销售成本或利润。尚未收货或存在差异的数量及价值继续保留在在途库存，直至受控差异处理生效。

调拨短少在调查期间继续保留在途数量和价值；确认损坏或丢失会改变库存价值，属于重大会计决定，须由对应三人会计组至少两人同意并经经理批准后，以追加调整流水和分录处理并提醒监察员。退回发出仓通过反向调拨流水和相应在途转回分录处理。未经确认的超收实物只记录为调拨隔离库存，不进入库存资产账册。

错误通过反向或补充分录更正，原分录永久保留。采购暂记和销售暂记不代表真实应付、应收或现金，首版也不承担收付款、银行、税务申报、工资或费用报销等完整企业总账职责，以控制范围并为未来总账集成保留清晰边界。

内部库存子账是库存会计事实源。每次分录入账时在同一事务中追加版本化账册导出事件，供外部总账及上下级仓库账册按游标拉取或由可替换适配器推送；外部系统只能回传接收状态和凭证号，不能覆盖内部分录。错误必须回到内部会计调整流程处理，再导出相应反向或补充分录。
