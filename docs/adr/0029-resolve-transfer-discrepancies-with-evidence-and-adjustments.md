# 调拨差异通过双边证据和追加调整解决

跨仓调拨实收与对应在途数量不一致时立即建立不可变差异记录，关联调拨、发运、收货、期望量、实收量及双方单据。发出仓与接收仓负责人分别提交说明、证据和建议处理方式；任何一方不能直接修改在途、接收或隔离数量。短少数量继续保留在途状态，超过对应在途数量的实物进入调拨隔离库存，不能销售、移位或计入接收仓库存资产。

允许的解决结果包括：延迟到货继续按原调拨分批收货；退回发出仓经双边确认后追加反向调拨流水及在途转回分录；来源识别错误的实物关联正确调拨；无法关联的超收实物另走正常入库审批。确认损坏或丢失会改变库存数量和价值，属于重大会计决定，须由对应三人会计组至少两人同意并经经理批准，系统同时提醒监察员，再生成追加的库存调整流水和会计调整分录。

经理只能在双边证据已经提交后批准业务解决结果；若一侧管理员暂停，则由既有替代审批规则提交该侧意见。所有决定、表决、证据、提醒、反向流水和调整分录永久关联原差异，不能覆盖原发运、收货或会计事实。该设计接受差异在解决前持续占用在途或隔离状态，以换取数量、责任和财务影响的完整可追溯性。
